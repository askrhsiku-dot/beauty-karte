# ✦ Karte | Beauty Salon

美容サロン向けの顧客カルテ管理アプリです。単一の `index.html` で動作するシングルページアプリで、サーバーやビルド作業は不要です。

## 主な機能

- 顧客一覧の検索・並び替え（最近来店順、50音順、No.順、来店回数順）
- 顧客ごとの基本情報・来店履歴・カルテメモの管理
- 来店記録の追加・削除と、累計金額・平均来店周期の自動計算
- アレルギー情報やタグによる注意事項の管理
- CSVファイルからの顧客一括登録
- 全データをCSVでバックアップ書き出し
- アプリ名・サブタイトルのカスタマイズ（設定パネル）

## 使い方

1. `index.html` をブラウザで開く（ローカルファイルとして直接開くか、簡易HTTPサーバーで配信）
2. 「＋ 新規登録」から顧客情報を登録、または「統計」タブのCSV取込機能で一括登録
3. 顧客カードをタップすると詳細パネルが開き、来店履歴やメモを編集できます

## データの保存について

入力したデータはブラウザの `localStorage` に保存されます。サーバーへの送信は行われません。

- 端末・ブラウザを変更するとデータは引き継がれません
- 「統計」タブの「全データをCSVで書き出す」機能で定期的にバックアップを取ることを推奨します
- CSV取込機能を使うと、バックアップから顧客情報を復元できます

## 技術構成
